# Blend_Mode

https://skia.org/docs/user/api/skblendmode_overview/#Blend_Mode

## Краткая практическая памятка

Чаще всего в реальной работе нужны такие режимы:

* **`SrcOver`** — обычное рисование поверх;
* **`Clear`** — стереть;
* **`Src`** — полностью заменить;
* **`SrcIn` / `DstIn`** — маска;
* **`SrcOut` / `DstOut`** — вычитание формы;
* **`Multiply`** — затемнение;
* **`Screen`** — осветление;
* **`Overlay`** — усиление контраста;
* **`Plus`** — свечение и накопление яркости;
* **`Color`** — перекраска с сохранением светотени.


## Все режимы

| Режим        | Что делает                                                                                     | Когда применять                                                |
| ------------ | ---------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| `Clear`      | Полностью очищает пиксели в зоне рисования. Результат становится прозрачным.                   | Когда нужно стереть часть изображения.                         |
| `Src`        | Полностью берет новый рисуемый пиксель и заменяет им старый.                                   | Когда нужно просто перезаписать пиксели без смешивания.        |
| `Dst`        | Полностью сохраняет старый пиксель, новый игнорируется.                                        | Почти не используется напрямую; полезен как служебный режим.   |
| `SrcOver`    | Рисует новый слой поверх старого с учетом прозрачности.                                        | Обычное стандартное рисование.                                 |
| `DstOver`    | Оставляет старый слой сверху, а новый как бы помещает под него.                                | Когда нужно дорисовать что-то под уже существующим содержимым. |
| `SrcIn`      | Оставляет новый слой только там, где уже есть старый.                                          | Для масок и вырезания source по форме destination.             |
| `DstIn`      | Оставляет старый слой только там, где есть новый.                                              | Для маскирования уже нарисованного изображения.                |
| `SrcOut`     | Оставляет новый слой только там, где старого нет.                                              | Когда нужно рисовать только снаружи существующей формы.        |
| `DstOut`     | Оставляет старый слой только там, где нового нет.                                              | Для вырезания формы source из destination.                     |
| `SrcATop`    | Накладывает новый слой только внутри области старого, сохраняя прозрачность старого.           | Для перекрашивания или заполнения существующей формы.          |
| `DstATop`    | Оставляет старый слой только внутри области нового.                                            | Для более редких задач композитинга и сложных масок.           |
| `Xor`        | Оставляет только непересекающиеся части обоих слоев. Общая область удаляется.                  | Для эффектов исключающего пересечения.                         |
| `Plus`       | Складывает яркость source и destination.                                                       | Для свечения, бликов, световых эффектов.                       |
| `Modulate`   | Перемножает source и destination с учетом прозрачности. Обычно затемняет.                      | Для затемнения и текстурного смешивания.                       |
| `Screen`     | Осветляет изображение, делая результат светлее исходных слоев.                                 | Для света, бликов, мягкого свечения.                           |
| `Overlay`    | Усиливает контраст: темное делает темнее, светлое светлее.                                     | Для более “сочного” и контрастного изображения.                |
| `Darken`     | Сравнивает два слоя и оставляет более темный цвет.                                             | Когда нужно сохранять только темные области.                   |
| `Lighten`    | Сравнивает два слоя и оставляет более светлый цвет.                                            | Когда нужно сохранять только светлые области.                  |
| `ColorDodge` | Сильно осветляет destination в зависимости от source.                                          | Для ярких бликов, засветки, эффектов яркого света.             |
| `ColorBurn`  | Сильно затемняет destination в зависимости от source.                                          | Для глубоких теней и драматичного контраста.                   |
| `HardLight`  | Резко осветляет или затемняет в зависимости от яркости source.                                 | Для жесткого контрастного наложения.                           |
| `SoftLight`  | Похож на `HardLight`, но работает мягче и деликатнее.                                          | Для мягкой коррекции света и контраста.                        |
| `Difference` | Показывает разницу между цветами source и destination. Одинаковые цвета дают темный результат. | Для спецэффектов, поиска различий, инверсий.                   |
| `Exclusion`  | Похож на `Difference`, но с более мягким и менее контрастным результатом.                      | Для мягких эффектов различия.                                  |
| `Multiply`   | Перемножает цвета, обычно заметно затемняя результат.                                          | Для теней, затемнения, печатных и текстурных эффектов.         |
| `Hue`        | Берет оттенок из source, а насыщенность и яркость из destination.                              | Для смены оттенка без потери светотени.                        |
| `Saturation` | Берет насыщенность из source, а оттенок и яркость из destination.                              | Для управления насыщенностью изображения.                      |
| `Color`      | Берет оттенок и насыщенность из source, а яркость из destination.                              | Для естественной перекраски объекта с сохранением объема.      |
| `Luminosity` | Берет яркость из source, а оттенок и насыщенность из destination.                              | Для переноса светлоты и тональной структуры.                   |
